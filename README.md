# Traversy Media Projects Collection

A collection of interactive JavaScript projects featuring HTML, CSS, and vanilla JavaScript implementations.

## Projects

### 1. **Blur Effect**
- Files: `blur.html`, `blur.css`, `blur.js`
- Description: Interactive blur effect with image manipulation

### 2. **Box Shadow**
- Files: `box.html`, `box.css`, `box.js`
- Description: Dynamic box shadow generator with interactive controls

### 3. **Double Click Heart**
- Files: `heart.html`, `heart.css`, `heart.js`
- Description: Double-click on an image to add hearts with counter tracking

### 4. **Image Gallery**
- Files: `image.html`, `image.css`, `image.js`
- Description: Interactive image gallery with navigation

### 5. **Live User Filter**
- Files: `liveuser.html`, `liveuser.css`, `liveuser.js`
- Description: Real-time user search and filtering functionality

### 6. **Login Form**
- Files: `login.html`, `login.css`, `login.js`
- Description: Interactive login form with form validation

### 7. **Image Slider**
- Files: `slider.html`, `slider.css`, `slider.js`
- Description: Carousel/slider component with navigation controls

## How to Use

Each project is self-contained with its own HTML, CSS, and JavaScript files. Simply open any `.html` file in your browser to view and interact with the project.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Font Awesome Icons (CDN)

## Getting Started

1. Clone or download this repository
2. Open any `.html` file in your web browser
3. Interact with the projects and explore the code
